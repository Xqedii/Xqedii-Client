[![Discord](https://img.shields.io/badge/Discord-Join-7289DA?logo=discord&logoColor=white)](https://discord.com/invite/6JQfeQEB4W)

**Xqedii Client is Minecraft Fabric 1.21.4 mod that adds tools useful for developers**

This client adds a variety of features that can be useful for developers when testing or creating servers. It includes functions such as reading NBT tags from the held item, GUI, or the currently hovered NBT.

> [!NOTE]
> This project is a work in progress and not finished!

-----

# Overview
The client includes a panel opened with the right Shift key, containing various selectable functions. For example, by clicking Main Hand NBT, the data from the item in your hand will be displayed in chat, allowing you to easily copy useful information such as the head texture, name, or lore.

This option allows you to read the NBT tags of the item currently in your hand.

## Targeted Entity NBT
This option allows you to read the NBT tags of the entity in front of you that you are currently targeting.

## GUI Item NBT
This option allows you to read the NBT tags of an item in a GUI. After enabling the option, you can open a GUI and click on any item to view its data.

## Installation
To install the client, download the repository and build the JAR file, or download it from Releases. Then, place the JAR file into the mods folder for Fabric 1.21.4.

> [!NOTE]
> For the panel to work properly, you need JDK 17+
